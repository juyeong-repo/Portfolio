# Portfolio

포트폴리오 프로젝트입니다.  
Next.js 14, TypeScript, Tailwind CSS를 기반으로 제작되었습니다.  

---

## 📂 프로젝트 구조
portfolio/
├─ src/
│ ├─ app/
│ │ ├─ layout.tsx # 공통 레이아웃
│ │ └─ page.tsx # 홈 페이지
│ ├─ components/ # 재사용 컴포넌트
│ └─ styles/
│ └─ globals.css # Tailwind 글로벌 CSS
├─ public/ # 이미지/정적 파일
├─ package.json
├─ tsconfig.json
└─ next.config.js


## 🚀 사용 기술 스택

- **Framework**: [Next.js 14](https://nextjs.org/) (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Package Manager**: npm
- **Runtime**: Node.js (v20+)
